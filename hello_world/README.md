*`0_Preprocessor`This project focuses on understanding the preprocessing stage of the C compilation process.
*`1_compiler`Task 1 focuses on compiling a C file into an object file without linking.
*`2_assembler`Task 2 focuses on generating assembly
*`3_name`Task 3 focuses on compiling a C File that creates an executable name
*`4_hello,puts`Task 4 focuses on writing a C program that prints a text on a new line by using the put function
*`5_printf.c`Task 5 focuses on writing a C program that prints text on a new line by using the printf function
*`6_size.c`Task 6 focuses on writing a C program that prints the size of various types on the computer it is compiled and run on.
